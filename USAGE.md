# English Talker — 使用说明

## 1. 应用简介

English Talker 是一款 AI 语音对话助手，你可以通过**语音**或**文字**与虚拟伙伴 "Luna" 进行自然流畅的对话。它支持语音识别输入、AI 智能回复、语音朗读回复，适合练习口语、获取信息或日常聊天。

## 2. 快速开始

### 2.1 安装 APK

在 [Releases 页面](https://github.com/thinkingandwritingrepeatedly/english-talker-app/releases) 下载最新版 APK，或直接使用项目根目录下的 `app-release.apk`。

> **注意**：APK 需要 Android 7.0 (API 24) 及以上版本。

### 2.2 配置 API Key

首次打开 App 时会自动弹出设置界面（也可点击右上角 ⚙ 进入）。

1. **API Key**（必填）：输入你的 DeepSeek API 密钥（可从 [platform.deepseek.com](https://platform.deepseek.com) 获取）
2. **Base URL**（可选）：默认对接 DeepSeek，可改为 OpenAI、通义千问等兼容接口
3. **模型**（可选）：默认 `deepseek-chat`
4. **输入语言**：选择语音识别的语言（影响识别准确率）

![设置界面示意]

### 2.3 开始对话

**语音对话：**
1. 点击中间的 **「▶ 开始对话」** 按钮
2. 授权麦克风权限（首次使用）
3. 开始说话，App 会实时显示识别文字
4. 停止说话约 2 秒后，Luna 会自动回复并朗读

**文字对话：**
1. 在输入框中输入文字
2. 点击 **「发送」** 按钮
3. Luna 的回复会显示在聊天列表中（如开启语音朗读则会同时朗读）

> **提示**：你可以在语音对话过程中随时打字输入，语音会自动停止并切换到文字模式。

## 4. 核心功能

### 4.1 语音对话

| 操作 | 说明 |
|---|---|
| 点击「开始对话」 | 启动语音识别，开始聆听 |
| 说话 | 实时显示识别文字（灰色为临时结果） |
| 停止说话 | 静音 2 秒后自动发送给 AI |
| 点击「停止」 | 手动结束本轮语音对话 |
| 连续对话 | AI 回复朗读完成后自动重新开始聆听 |

### 4.2 文字对话

| 操作 | 说明 |
|---|---|
| 点击输入框 | 输入文字消息 |
| 点击「发送」 | 发送文字给 AI |
| 发送后 | AI 回复显示在聊天列表中 |

### 4.3 语音朗读开关

底部开关控制 AI 回复是否朗读：

- **开启**（默认）：AI 回复会自动朗读
- **关闭**：仅显示文字回复，不朗读

> 语音对话模式下，关闭朗读后回复完成会保持 `listening` 状态继续聆听。

### 4.4 重置对话

点击右上角 **↺** 重置按钮：
- 清空全部对话历史
- 停止语音活动
- 回到初始状态

### 4.5 设置

点击右上角 **⚙** 设置按钮：

| 设置项 | 说明 |
|---|---|
| API Key | 必填。AI 服务认证密钥 |
| Base URL | 可选。API 服务地址，默认 DeepSeek |
| 模型 | 可选。模型名称 |
| 输入语言 | 语音识别语言。可选 en-US / zh-CN / ja-JP / ko-KR |

## 5. 状态说明

App 会通过按钮样式和头像动画反映当前状态：

| 状态 | 按钮 | 头像动画 | 说明 |
|---|---|---|---|
| 待机 | 「▶ 开始对话」绿色 | 紫色辉光缓慢脉动 | 等待操作 |
| 准备中 | 不可操作 | 紫色辉光快速脉动 | 初始化语音引擎 |
| 聆听中 | 「■ 停止」红色 | 绿色辉光快速脉动 | 正在听你说话 |
| 思考中 | 不可操作 | 金色旋转光环 | AI 正在生成回复 |
| 朗读中 | 不可操作 | 紫色辉光快速脉动 + 嘴巴动画 | AI 正在朗读 |
| 文字模式 | — | — | 设备不支持语音，仅可打字 |

## 6. 常见问题

### Q: 点击「开始对话」没有反应？

请检查：
1. 是否已正确填写 API Key（点击 ⚙ 查看）
2. 是否已授予麦克风权限
3. 设备是否支持语音识别（部分模拟器不支持）

### Q: 语音识别不准确？

1. 在设置中确认「输入语言」选择了正确的语言
2. 在安静的环境中使用
3. 说话清晰，语速适中

### Q: AI 回复太慢？

1. 检查网络连接
2. 可以尝试更换更快的模型（如 `deepseek-chat` 或 `gpt-4o-mini`）
3. API 免费额度可能已用完

### Q: 怎样切换其他 AI 模型？

在设置中修改 Base URL 和 Model 即可。支持任何 OpenAI 兼容的 API：

| 服务 | Base URL |
|---|---|
| DeepSeek | `https://api.deepseek.com/v1` |
| OpenAI | `https://api.openai.com/v1` |
| 通义千问 | `https://dashscope.aliyuncs.com/compatible-mode/v1` |

### Q: 语音朗读没有声音？

1. 检查底部「语音」开关是否开启
2. 检查手机媒体音量是否打开
3. 部分模拟器不支持 TTS

### Q: 如何重置所有设置？

App 设置存储在本地，卸载重装或清除应用数据即可恢复默认设置。

## 7. 隐私说明

- 语音识别在**设备端**实时处理，识别的文字通过 API 发送给 AI 服务
- API Key 仅存储在本地设备
- 对话历史存储在内存中，关闭 App 即消失
- 不会收集任何个人数据
